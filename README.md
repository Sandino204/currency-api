# Currency converter

## Description
- Uma api que permite criar uma moeda, deletar e editar uma moeda
- Permite ao usuario coverter o valor da moeda com moedas existentes no banco de dados y

## Commands

### Test the API
```
    npm run test
```

### Run the API
```
    docker compose up
```
